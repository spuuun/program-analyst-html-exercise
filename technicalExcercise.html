  • Under Submission Manager, open Configuration Settings
  • In Settings, go to Activity and select Types
  • NOTE: At this point, it is a good idea to open a current activity in a new window as a reference in case you need to mirror the settings.
  • Click the (+) New Type
    • Enter the information
    • Name – title of activity, not displayed
    • Caption – display name
    • Display order – add on at the end
    • Check – Display as Tab on List view under Parent
    • If it says “Other Activity”, go back and select “Display as Tab”
  • Save
    • Permissions
      • Copy the Restrict Access To permissions from the other open activity
      • Reviewer, Staff, System Admin
      • Available to Statuses – when you want it to show up
      • Submitted, approved, declined, completed
      • Available Grant Types – where to show up
      • Fast change
  • Save
  • Select Custom Fields in the left menu bar
  • Custom fields – create a title bar new custom fields
    • Number of columns – 1
    • Columns under title bar
      • If you want dynamic radio buttons, you need a title bar with minimum of 1 column above the radio button
    • Tabbed section to make additional tabs in activity
  • New Activities do not get versioned, only their custom fields
